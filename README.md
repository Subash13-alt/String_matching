# String Matching Algorithms

A Streamlit application that demonstrates and compares three string matching algorithms:

* Naive Search
* Knuth-Morris-Pratt (KMP)
* Rabin-Karp

## Live Demo

https://stringmatching-5p8bg64c2nrpd57teidqzy.streamlit.app/

## Features

* Pattern searching in text
* Match position detection
* Comparison of algorithm performance
* Interactive Streamlit interface

## Technologies Used

* Python
* Streamlit

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
